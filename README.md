# Aplicação de PDI para Análise de Envelhecimento Facial

Este projeto utiliza técnicas de Processamento Digital de Imagens (OpenCV)
aliadas a Redes Neurais Convolucionais (TensorFlow/Keras) para analisar sinais
visuais associados ao envelhecimento facial.

## Técnicas Utilizadas
- Detecção facial (Haar Cascade)
- Detecção de bordas (Canny)
- Análise de textura (Laplaciano)
- CNN para classificação

## Objetivo Acadêmico
Projeto desenvolvido para a disciplina de Processamento Digital de Imagens.
